## ElaAdmin for Laravel

> Implementação do template ElaAdmin em um projeto PHP utilizando Laravel Framework


### Tecnologias utilizadas: 
#### Frontend
- HTML5
- CSS3
- Bootstrap
- JavaScript
- JQuery
- ElaAdmin Template

#### Backend
- PHP 7.2
- Laravel 6.2

#### Bibliotecas
- Laravel Collective

---

### Instalação

#### Backend
- $ composer install
